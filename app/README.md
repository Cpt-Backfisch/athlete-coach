# athlete.coach — React-App (`app/`)

Dieser Ordner enthält die neue React-Frontendapp (Vite + React + TypeScript), die im Rahmen der Phase-1-Migration parallel zur bestehenden `index.html` im Root aufgebaut wird.

Entwicklungsserver starten: `npm run dev` (startet auf http://localhost:5173)

Produktions-Build erstellen: `npm run build` (Ausgabe landet in `app/dist/`)
